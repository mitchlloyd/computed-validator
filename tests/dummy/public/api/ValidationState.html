Class: ValidationState

ValidationState

new ValidationState(allErrors, translate)

Whenever a validation rule runs, it returns a new validationState instance.

A ValidationState instance can have pending promises that may return errors in the future. However, a ValidationState will not change when these promises resolve. Rather it exports nextValidationState, a function that can will asynchronously retrieve a new validaiton state.
